# Wohnungs-Tool · Klickdummy

Mobile-first Klickdummy fuer ein persoenliches Wohnungssuche-Tool. Mockup, kein Backend — alle Daten sind Demo.

## Was ist das

Jonathan baut sich ein eigenes Cockpit fuer die Wohnungssuche in Frankfurt. Ziel: Online-Inserate (IS24, Kleinanzeigen, WG-Gesucht) und Off-Market-Aktivitaeten (Briefe an Hausverwaltungen, Aushaenge an schwarzen Brettern, Anrufe bei Maklern, Vitamin-B-Anfragen) als **gleichwertige Konversationen** managen.

## Was du sehen kannst

- **Today-Stream:** was heute zu tun ist, mit Hot / Termine / Heute / Spaeter
- **Karte:** Frankfurt mit Pins (Inserate, Aushang-Spots, Hausverwaltungen, Makler, Vitamin-B)
- **Inbox-Triage:** neue Inserate aus mehreren Quellen, 1-Tap-Triage
- **Pipeline:** alle Faelle filterbar nach Konfig (Solo / 2er / 3er / 4er-WG), Bezirk, Status
- **Stats:** Konsistenz + Prognose
- **Profil:** Bewerbungs-Pack (Schufa, Gehaltsnachweis, etc.), Vitamin-B-Liste, Bezirke + Kriterien

## Was funktioniert im Mockup

- Tab-Wechsel zwischen Today / Karte / Pipeline / Stats
- Filter-Pills (Konfig, Bezirk, Status, Kanal)
- Drilldown-Sheets bei Tap auf Cards
- AI-Antwort-Vorschlag im Mail-Drilldown
- Schnellaufnahme-Bar oben mit Voice + Tag-Buttons
- Onboarding-Wizard (4 Schritte)
- Aushang-/Brief-Generator
- Light + Dark Mode (auto via System)

## Was NICHT funktioniert (Mockup)

- Echter Versand von Mails / Briefen
- Live-Daten (alles ist statische Demo)
- Account-System / Login
